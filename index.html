Sumitomo NY Real Estate Sales
  • ASP.NET MVC 5
  • Entity Framework
  • SQL Server
  • JavaScript, HTML5, CSS3
  • jQuery
  • Bootstrap
  • Google Maps JavaScript API

A multilingual real estate web application built using the ASP.NET MVC5 framework. This was created and managed by a small team using Git as the version control system.

  • Designed the SQL Server Database to store data for Listings and Agents
  • Created a C# Library that imports data from Multiple Listing Service (MLS), maps it to the database entity object, and then stores it to the database using the Entity Framework
  • Created responsive web pages using Bootstrap
  • Created dynamic pages that gets data from the database asynchronously using jQuery
  • Added support for other languages
